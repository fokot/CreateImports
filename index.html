CreateImports by fokot

Eclipse plugin that takes a java project and replaces fullyqualified names by an import and simple name. Replacement for hybridlabs-beautifier.

Useful when generating java sources. You can generate full java names e.g. com.fokot.CreateImports so no collisions arise. After generation you can use this plugin to simplify fully qualified names and create imports for it. It takes java source files in Eclipse, and simplifies the fully qualified names if possible. We use in our own plugin which generates code.

There project which does the same called hybridlabs-beautifier, but it has many dependecies and skips the files which have a wildcard import in them e.g. import com.fokot.*. And we even create imports for static method calls.

Install

Copy into eclipse/plugins directory or use as a jar in your project.

To run CreateImprotsActrionOn you sources do following: 
CreateImportsOperation ci = new CreateImportsOperation(javaElement);
ci.execute(progressMonitor);
